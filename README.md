Abstract:  Specific target sounds can be distinguished from normal environment sounds using machine learning classification algorithms.  The identification can produce an alert and audio recording for further action.  Use of additional video equipment, with time stamps, can connect video of the circumstances with each event.  The target sound for this specific project was a reciprocating saw cutting a steal pipe (as in catalytic converter theft).  The environment was an apartment complex parking lot with typical maintenance and tenant noises.  Several machine learning SKLearn classifiers were able to distinguish the target sound from typical electrical/mechanical equipment operating in the environment.  Algorithms using Fourier transformed signals produced the most accurate results.
